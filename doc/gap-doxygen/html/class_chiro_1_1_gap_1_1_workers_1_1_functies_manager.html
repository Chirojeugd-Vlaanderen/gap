Functie Chiro.Gap.Workers.FunctiesManager.Verwijderen ( Functie  functie,
bool  forceren 
)

Verwijdert een functie (PERSISTEERT!)

Parameters:
functieTe verwijderen functie, inclusief groep, leden en groepswerkjaar leden
forcerenIndien true wordt de functie ook verwijderd als er dit werkjaar personen met de gegeven functie zijn. Anders krijg je een exception.
Retourneert:
Null als de functie effectief verwijderd is, anders het functie-object met aangepast 'werkjaartot'.

Als de functie geen leden meer bevat na verwijdering van die van het huidige werkjaar, dan wordt ze verwijderd. Zo niet, wordt er een stopdatum op geplakt.
