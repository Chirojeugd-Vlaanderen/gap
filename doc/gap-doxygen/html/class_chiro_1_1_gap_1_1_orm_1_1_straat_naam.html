Gedetailleerde Beschrijving

No Metadata Documentation available.

Instantieert een Straat-object dat zorgt voor samenwerking met Entity Framework.

Een straat heeft geen versie (timestamp) in de database. Dat lijkt me ook niet direct nodig voor een klasse die bijna nooit wijzigt.

Het feit dat er geen timestamp is, wil wel zeggen dat 'concurrencygewijze' de laatste altijd zal winnen.
