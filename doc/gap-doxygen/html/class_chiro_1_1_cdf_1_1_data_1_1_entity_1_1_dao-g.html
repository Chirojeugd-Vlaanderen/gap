template<TEntiteit, TContext>
class Chiro::Cdf::Data::Entity::Dao< TEntiteit, TContext >

Algemene implementatie van IDao; generieke CRUD-operaties voor een DAO-object.

Template Parameters:
TEntiteitType van entiteit waarvoor deze DAO een repository is
TContextType van de objectcontext die de entiteiten zal ophalen/bewaren

Alle entity's die de DAO oplevert, moeten gedetacht zijn. Dit kan via MergeOption.NoTracking, of via Utility.DetachObjectGraph.
