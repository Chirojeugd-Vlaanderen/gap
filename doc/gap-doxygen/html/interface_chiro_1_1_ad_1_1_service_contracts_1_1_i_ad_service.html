Vraagt aan Active Directory om een account aan te maken met rechten om GAP te gebruiken. Als de persoon in kwestie al een account heeft, worden de rechten uitgebreid.

Parameters:
adNrAD-nummer van de persoon die een login moet krijgen
voornaamVoornaam van de persoon die een login moet krijgen
familienaamNaam van de persoon die een login moet krijgen
mailadresMailadres van de persoon die een login moet krijgen
Retourneert:
Als alles goed gaat: de loginnaam. Het kan zijn dat die al bestond - in dat geval worden de GAP-rechten toegekend. Ofwel werd hij aangemaakt. In beide gevallen krijgt de persoon in kwestie een mailtje zodat hij of zij op de hoogte is.

Dit is een voorbeeld van hoe deze service aangeroepen kan worden: 

                string login;
                ServiceClient client = null;
                try
                {
                    client = new ServiceClient();
                    login = client.GapLoginAanvragen(Int32.Parse(AdNrTextBox.Text), VoornaamTextBox.Text, NaamTextBox.Text, MailadresTextBox.Text);
                }
                catch
                {
                    // Fout afhandelen
                }
                finally
                {
                    if (client != null)
                    {
                        // Altijd de client afsluiten!
                        client.Close();
                    }
                }
