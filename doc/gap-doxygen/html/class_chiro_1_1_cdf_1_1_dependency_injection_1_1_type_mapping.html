Gedetailleerde Beschrijving

Mapping between a requested type and the type to build by the object builder. Requested type is usually an interface whereas typeToBuild is a concrete implementation of that interface. If this type mapping is not specified and the requested type is an interface, object builder will not know how to create a new instance of that type.
