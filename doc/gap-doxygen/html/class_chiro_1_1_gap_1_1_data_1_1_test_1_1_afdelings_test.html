Gedetailleerde Beschrijving

Data-Access tests voor afdelingen en alles wat daarrond hangt.

Deze tests werken op een testgroep in de database. Het script om de testgroep te maken, zit in SVN: https://develop.chiro.be/subversion/cg2/trunk/database/TestData/TestData/TestGroepMaken.sql

De IDs nodig voor de test zitten in de settings van dit project.
